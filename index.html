自动下载 Java

检测到没有 Java 时自动从 Adoptium 下载 Java 21，不需要手动安装，也支持指定自己的 Java 路径。

⚡

多线程下载

16 线程并发下载游戏文件，失败自动重试，比单线程快好几倍，装完就能玩。

📦

本地化存储

所有文件存在启动器旁边的 .minecraft 目录，结构和 PCL2 一样，不会乱写系统目录。

🗂️

版本管理

已安装的版本一目了然，点一下直接启动，不用每次重新安装。

💾

配置保存

用户名、Java 路径、上次选的版本都会自动保存，下次打开不用重新填。

🎨

Fluent 界面

基于 PyQt-Fluent-Widgets 构建，深色主题，看着舒服，用着顺手。

快速开始
三步开始游戏
不需要提前装任何东西。
1

下载启动器

把启动器放到任意目录，双击运行。

2

选版本安装

3

启动游戏

填好用户名，点"启动已安装版本"，进游戏。
