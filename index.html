常见问题

有疑问？这里可能有你要的答案。

需要自己安装 Java 吗？

不需要。FenCH 会根据 Minecraft 版本自动检测并下载对应的 Java（8 / 17 / 21），存放在启动器目录的 .Java 文件夹中。

支持哪些登录方式？

支持三种：微软正版账号（OAuth）、第三方皮肤站（authlib-injector，如 LittleSkin）、离线账号。Token 过期会静默刷新，不会弹出烦人的错误提示。

联机功能怎么用？

在"更多 → 联机"页面创建或加入房间，基于 UDP 打洞实现跨网络连接，仅限使用 FenCH 启动器的玩家之间使用，无需第三方工具。

聊天功能需要什么条件？

需要先在账号页面登录 Minecraft 账号（正版、第三方或离线均可）。消息通过本地文件共享，所有使用同一启动器目录的用户可以互相看到消息。
